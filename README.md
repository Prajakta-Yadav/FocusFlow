FocusFlow – Pomodoro Timer Web App

FocusFlow is a simple productivity web application based on the Pomodoro Technique, designed to help users stay focused during work or study sessions. The application runs a 25-minute countdown timer and alerts the user when a focus session is completed.

This project demonstrates the implementation of frontend development fundamentals using HTML, CSS, and JavaScript.

Technologies Used

HTML – Used for creating the structure and layout of the application.

CSS – Used for designing the user interface with a modern and responsive style.

JavaScript – Used to implement the timer functionality and interactive behavior.

Features

⏱ Pomodoro Timer (25 minutes) countdown for focused work sessions.

▶ Start Button – Starts the timer.

⏸ Pause Button – Temporarily stops the timer.

🔄 Reset Button – Resets the timer to the initial value.

🔔 Audio Alarm – Plays an alarm sound when the Pomodoro session is completed.

⚠ Alert Notification – Displays a popup alert when the session ends.

🎵 Stop Alarm on Alert Confirmation – When the user clicks OK on the alert message, the alarm sound stops and the timer resets for the next session.

JavaScript Implementation

The timer functionality is implemented using:

setInterval() / setTimeout() for countdown behavior

Custom timer() function to manage time updates

DOM manipulation for updating the timer display

Audio control using JavaScript to play and stop the alarm sound

Learning Outcomes

Through this project, I learned:

DOM manipulation using JavaScript

Implementing timers with setInterval()

Handling user interaction with buttons

Integrating audio alerts in web applications

Building a simple productivity tool using core web technologies
